# Security Policy

## Supported Versions

Currently, only the latest version of SAB is supported with security updates.

| Version | Supported |
| ------- | ------------------ |
| 2.x | :white_check_mark: |
| < 2.x | :x: |


### What to include in your report:

- Type of issue (e.g., buffer overflow, SQL injection, cross-site scripting, etc.)
- Full details of the vulnerability
- Step-by-step instructions to reproduce the issue
- Potential impact of the vulnerability

### Our response process:

1. We will acknowledge receipt of your report within 48 hours.
2. We will investigate the issue and confirm the vulnerability.
3. We will work on a fix and coordinate a release date with you.
4. We will credit you for your discovery in our release notes (unless you prefer to remain anonymous).
